Любая сущность JS - это объект. Объект набор свойств.
PascalCase - типы и классы. DB_PASS - значения известны до запуска приложения и не меняются.
camelCase - все остальные переменные.
Примитивные типы: string,boolean,number,null, undefined, symbol
Cсылочный тип:object(объект).
Функция блок кода который можно выполнять многократно.
Фунции: именнованная, присвоена переменной, анонимной, аргументом при вызове другой функции,
значением свойства (метода) объекта.
Колбэк функции - функции вызываемые внутри других функций.
Анонимные функции "("аргументы если они есть")=> {"тело можно и без скобок если изи"}".
Инструкция заканчивается ";" таким образом можно отличить инструкцию от выражения.
Выражение может быть передано в функцию в качестве аргумента,а инструкция не может.
Массив - это объект с цифровыми именами свойств.
Промисы позволяют обрабатывать отложенные во времени события. Промис - это обещание предоставить результат позже.
ASYNC/AWAIT - специальный синтаксис для упрощения работы с промисами.
Возвращает Промис. Их следует помещать в блоки try catch для корректной работы.
